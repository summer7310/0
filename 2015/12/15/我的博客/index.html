display: table;
display: table-cell;
vertical-align: middle;
  • Everything in this table cell div is centred
  • It can dynamically change to any height based on content (try by changing your browser font size)
  • Doesn’ t get cut off when there isn't enough room
  • Doesn’ t work in Internet Explorer
  • Uses 2 extra divs
Any object, with relative positioning

positoin: absolute; top: 50%; height: 240px; margin-top: (negative half the height);

  • Works in all browsers
  • Fixed height object (can be ems)
  • Gets cut off when there isn't enough space
  • Doesn't require additional objects
float: left;
height: 50%;
margin-bottom: -120px;

position: relative; clear: left; height: 240px;

  • Works in all browsers
  • Fixed Height (can be ems)
  • Doesn't get cut off when there isn't enough space
  • Uses 1 extra empty div (not too bad)
Any object, with relative positioning

top: 0; bottom: 0; left: 0; right: 0;
position: absolute; margin: auto;
height: something; width: something;

  • Must have a fixed width and height
  • This doesn't work in Internet Explorer
  • Gets cut off when there isn't enough space
Single Line

line-height: 10em; height: 10em;

  • Only used to centre a single line of text
  • Container must have fixed height (can be em, can't be %)
  • (very useful for making buttons)
  • Works in all browsers
  • Doesn't get cut off when the isn't enough space
  • When there is more than a single line, it breaks badly
